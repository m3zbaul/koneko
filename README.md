## What's included:

- [x] React 16 (16.5.2)
- [x] Webpack 4 (4.20.2)
- [x] Webpack dev server
- [x] SASS support
- [x] Seperate stylesheet
- [x] PostCSS
- [x] Linting (ESLint, StyleLint)
